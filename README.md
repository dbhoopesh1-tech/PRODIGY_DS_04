# 📊 Titanic Dataset - Data Cleaning & Exploratory Data Analysis

## 📌 Objective
Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to identify patterns, handle missing values, and gain meaningful insights.

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
The project uses the Titanic dataset containing passenger information such as:
- Age
- Gender
- Fare
- Cabin
- Embarked Port

## 🔍 Data Cleaning
- Checked dataset structure and data types
- Identified missing values
- Filled missing Age values using the mean
- Replaced missing Cabin values with "Unknown"
- Removed rows with missing Embarked values
- Checked for duplicate records

## 📈 Exploratory Data Analysis
- Generated descriptive statistics
- Analyzed passenger age distribution
- Compared passenger counts by gender
- Explored the relationship between Age and Fare
- Visualized data using charts and plots

## 📊 Visualizations
- Age Distribution Histogram
- Gender Count Plot
- Age vs Fare Scatter Plot
